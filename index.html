About Preferences

User preferences let individuals opt out of specific delivery channels. The service respects these flags before dispatching any notification.

If a channel is disabled and a notification targets that channel, it will be silently skipped rather than queued.

API Endpoint
POST /users/:userId/preferences-bulk

How Webhooks Work

When a notification reaches a terminal state (SENT or FAILED), Notipy fires an HTTP POST to your registered endpoint.

If a secret is set, the request includes an X-Notipy-Signature header for HMAC-SHA256 verification.

Payload Shape
{
  "event": "sent",
  "notification_id": 42,
  "user_id": "user_web_123",
  "channel": "email",
  "timestamp": "2024-01-01T00:00:00Z"
}
